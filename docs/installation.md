# Installation

## Install Package

```bash
npm install paymcp @modelcontextprotocol/sdk
```

## Development Install

```bash
git clone https://github.com/yourusername/paymcp-ts.git
cd paymcp-ts
npm install
npm run build
```

## Requirements

- Node.js 18+
- TypeScript 5+

## TypeScript Config

```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "NodeNext",
    "moduleResolution": "NodeNext",
    "strict": true
  }
}
```
